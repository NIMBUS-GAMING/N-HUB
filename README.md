# N-HUB

N-HUB is a powerful Windows configuration tool specifically designed for Nimbus custom keyboards.

## Features
- Reliable configuration utility for Nimbus custom keyboard users.
- Helps to manage and customize keyboard settings effortlessly on Windows.

## Getting Started
